# CV-Online
Currículum Vitae online, interactivo.

Este sitio cuenta con todos los ítems que componen un Currículum vitae pero con varios puntos interactivos:

-Cambio de imagen de perfil con evento OnClick
-Dark Mode
-Modal para datos personales (ícono a la derecha del nombre)
-Dropdown en el menun lateral
-Enlaces directos a Whatsapp, llamada telefónica y correo electrónico
-Acordeones para Idioma y Skills
